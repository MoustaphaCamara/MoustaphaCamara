I mostly build with [![PHP](https://img.shields.io/badge/php-%23777BB4.svg?logo=php&logoColor=white)](#) and [![TypeScript](https://img.shields.io/badge/TypeScript-3178C6?logo=typescript&logoColor=fff)](#).

I mainly contribute to projects around the <a href="https://github.com/elabftw/elabftw">eLabFTW</a> ecosystem, a free and open-source electronic laboratory notebook for researchers.

I also contribute to [Deltablot](https://www.deltablot.com/) projects around automation, containerization, shell tooling, Docker images, and API clients for the eLabFTW ecosystem.

You can support the eLabFTW project by becoming a [GitHub Sponsor](https://github.com/sponsors/NicolasCARPi/) or through [Liberapay](https://liberapay.com/NicolasCARPi/).
